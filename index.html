PROJECT DIBUAT

Sistem Informasi Perpustakaan Unmuh

Aplikasi Sistem Informasi Perpustakaan ini dibuat untuk menampilkan data-data yang tersimpan pada perpustakaan. Contohnya data mahasiswa, buku, peminjaman dll. Aplikasi ini dibuat dengan menggunakan framework LARAVEL dan framework BOOTSTRAP. Aplikasi ini juga di lengkapi dengan fitur Registrasi dan Login

Sistem Informasi Rumah Sakit

Aplikasi Sistem Informasi Rumah Sakit merupakan tugas dari kampus, mata kuliah Basis Data, yang dimana aplikasi ini dibuat menggunakan bahasa pemrograman PHP. fitur-fitur pada aplikasi ini diantaranya Registrasi, Login, dan CRUD

Profil Perusahaan

Project profil perusahaan, sebuah project landing page yang dibuat untuk menjelaskan mengenai suatu perusahaan secara ringkas. Project ini dibuat dengan menggunakan bantuan framework CSS yaitu BOOTSTRAP.
